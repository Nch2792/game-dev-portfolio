# Game Development Portfolio

## Description
This is my game development portfolio made customed through standard front-end web development. This project showcases some big game projects that I have done in the past through my time in college. The purpose of this portolio is to practice HTML, CSS, and JavaScript, with the help of Codex. While AI was used to help structure the website, the adjustments and optimization of the website was completed manually. My hope is to show anyone, whether it be an employer, fellow game developer, or random person online, my skills and achievements in the past years. In addition, this project is one of many projects that I want to make to pursue my passion in software engineering, continuing to refine my skills, and learn new skills.
